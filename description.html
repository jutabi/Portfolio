주요 기능


비회원 사용자


  간판 검색

    ->간판 사진을 업로드하여 가게의 정보를 검색할 수 있습니다.


회원 사용자


  간판 검색

    ->간판을 검색하고 별점과 한줄평을 작성할 수 있습니다.

  검색 기록

    ->자신이 매긴 별점과 한줄평, 가게 정보를 담고있는 가게 기록을 확인할 수 있습니다.

  즐겨찾기

    ->검색기록을 즐겨찾기에 추가해 맛집 저장 노트로 활용할 수 있습니다.

수행 내용 (간판 검색시)


1. 이용자가 간판 사진을 업로드 -> 간판의 영역(ROI)을 지정 -> OCR하여 상호명을 추출


2. 사진의 GPS 정보를 이용해 네이버 Reverse Geocoding API에 지번주소를 요청합니다.


3. 지번주소에서 '행정동'을 추출하고 OCR한 상호명과 함께 네이버 검색 API에 가게 정보를 요청합니다.


4. 가게 정보와 네이버 지도를 검색 결과 View에 전달합니다.


5. 회원의 경우 검색한 가게에 대한 별점과 한줄평을 매길 수 있습니다.


6. 회원의 경우 검색 결과와 별점, 한줄평을 DB에 저장하여 검색 기록을 만듭니다.
